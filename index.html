Projects

Sarbanash

Sarbanash is a survival-style pixel game where the player must dodge and escape waves of enemies. Developed in Godot, it includes smart enemy movement, player health management, hit hurt system and rpg based movement. The game emphasizes real-time responsiveness and scene control using GDScript.

Kisor

Kisor is a 2D Visual Novel built using the Godot Engine. The game showcases branching dialogue options, story telling, and scene transitions using GDScript. Designed and published as an independent project on Itch.io, with the help of my friends for sprites and audio.

Review Project

Golden Pledges

Golden Pledges is a modern web application for managing and tracking pledges. Built with React and deployed on Vercel, it features a clean, intuitive interface for creating pledges, tracking progress, and managing commitments with real-time updates and responsive design.

Online Quiz

Online Quiz is an interactive web-based quiz platform built with modern web technologies and deployed on Render. Users can take engaging quizzes on various topics with features like timer functionality, instant score tracking, progress saving, and responsive design for all devices.
